## Sharp TV

This module will connect to a Sharp TV, it was build around a PN-LE901, but should work with most Sharp TV's/monitors with a network connection.

**Available commands for Sharp TV's**
* Quick Power ON
* Quick Power OFF
* Standby Power ON
* Standby Power OFF
* Input Select TV
* Input Select
* AV Mode Selection
* Set Volume
* View Mode
* Channel (Analog)
* Channel (Digital <10000)
* Channel (Digital >10000)
* Channel Up
* Channel Down
* Myte Audio ON
* Mute Audio OFF
* Toggle Mute Audio
* Surround Audio ON
* Surround Audio OFF
* Toggle Surround Audio
* Toggle Closed Captions
* Sleep Timer

**Not Implementet**
* Position
* Device Name
* Model Name
* Software Version
* IP Protocol Version
* Remote Control Buttons
